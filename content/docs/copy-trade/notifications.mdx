# LP Notifications with MetEngine
<img src="../../../n2.png" alt="A wild meteor" width="500" height="500"></img>

MetEngine now supports LP notifications, alerting you whenever specific wallets take LP positions. 

Follow these simple steps to set up the bot and start receiving LP notifications :
## **Step 1: Start the Bot**
Go to `/start` and click on the `Auto Trade` button to initiate the process.
<img src="../../../cp1.png" alt="A wild meteor" width="500" height="500"></img>
## **Step 2: Add a Wallet**
Read the instructions displayed and click the `Add Wallet` button to add a wallet.

<img src="../../../cp2.png" alt="A wild meteor" width="500" height="500"></img>
## **Step 3: Enter Wallet Details**

Add the wallet using the following format:  
**`Wallet_Name WalletAddress`**  
For example: `AlphaWallet 5F3sa2TJ...`

<img src="../../../cp3.png" alt="A wild meteor" width="500" height="500"></img>
## **Step 4: Stop Copy Trading**
Click on the `Stop Copy trade` button to stop copying trades from the primary wallet.<br/>  
This will disable copy trading but allow you to continue receiving LP position notifications from added wallets.

<img src="../../../cp5.png" alt="A wild meteor" width="500" height="500"></img>
## **What Notifications Will You Receive?**

Once set up, you will receive LP notifications for the wallets you’ve added. The notification details will include:

- **Wallet Name** 
- **Pool Name** with a link to Meteora
- **Token Amount** with the value in USD
- **Fees Claimed**
- **Strategy type**
- Links to **Dexscreener** and **GmGn**
